# Install GNU CLISP

## Ubuntu/Debian
```bash
sudo apt update
sudo apt install clisp
```

## Arch
```bash
sudo pacman -S clisp
```

## macOS
```bash
brew install clisp
```

## Windows
1. Download installer from [clisp.org](https://clisp.org)
2. Run installer
3. Add to PATH

## Verify Installation
```bash
clisp --version
```

## Start REPL
```bash
clisp
```
